# Fermyon VSCode Themes

VSCode themes, made with love from Fermyon.

## Installation

```
ext install fermyon.vscode-themes
```

Or, install from source:

```
vsce package
```

Then right-click the .vsix file in VS Code and select "Install Extension VSIX"
